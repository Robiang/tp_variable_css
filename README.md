# DevDash UI Kit

##  Description
Projet de dashboard admin réalisé avec HTML et CSS.

L’objectif est de construire un **design system basé uniquement sur les variables CSS**.

---

##  Technologies utilisées
- HTML5
- CSS3 (Variables CSS)

---

##  Fonctionnalités
- Design system avec variables CSS
- Thème dark
- Thème alternatif
- Dashboard responsive
- Composants réutilisables (cards, buttons, sidebar)

---

##  Responsive
Le dashboard s’adapte aux écrans mobiles (layout en colonne).
